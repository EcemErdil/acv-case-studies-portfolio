# Dashboard Guide

This document explains the structure of the Power BI dashboard used in this project.

The dashboard transforms analytical outputs into operational insights for electricity distribution monitoring.

---

## 1. Executive Overview

The executive overview page provides a summary of key metrics.

Displayed indicators include:

- total records analyzed
- risky records
- high-risk installations
- average risk score
- anomaly type distribution

This page provides a high-level operational overview.

---

## 2. Regional Risk Analysis

This page focuses on geographical anomaly patterns.

Visualizations include:

- anomaly distribution by province
- anomaly ratios by district
- subscriber group risk analysis
- meter brand anomaly comparison

This analysis helps identify regional anomaly concentrations.

---

## 3. Time Trend Analysis

The time trend page analyzes anomaly behavior over time.

Key visualizations include:

- daily anomaly counts
- rolling risk score trends
- anomaly type trend analysis

This allows analysts to detect periods of abnormal behavior.

---

## 4. Technical Signal Diagnostics

This page focuses on technical indicators related to electricity measurements.

Indicators include:

- phase zero anomalies
- voltage missing consumption records
- reactive energy ratio patterns
- negative or zero consumption values

These signals may indicate measurement infrastructure issues or equipment faults.

---

## 5. Case Investigation

The case investigation page allows detailed analysis of individual installations.

Users can inspect:

- consumption trends
- current behavior
- phase imbalance signals
- anomaly indicators
- risk score evolution

This page supports targeted operational investigation.

---

## 6. Drill-Through Analysis

The dashboard also provides drill-through capability.

Users can select an installation and access detailed technical signal tables and time-series analysis for deeper inspection.
