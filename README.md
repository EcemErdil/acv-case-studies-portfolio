# ACV Analytics Case Studies Portfolio

This repository contains three end-to-end analytics case studies completed as part of the ACV training program.

## Included Projects

### 1. Credit Card Customer Analytics
Folder: `case_study_01_credit_card_customer_analytics`

This project focuses on customer segmentation, profitability analysis, dashboard design, and business insight generation using Advanced Excel and Power Query.

### 2. Energy Consumption, Customer Segmentation & Collection Risk Analysis
Folder: `case_study_02_energy_consumption_customer_risk_analytics`

This case study analyzes energy consumption behavior, customer segmentation, and collection risk indicators to support decision-focused retail energy operations.

### 3. Electricity Consumption Anomaly Detection and Operational Risk Analysis
Folder: `case_study_03_electricity_consumption_anomaly_analysis`

This project combines Python, SQL, and Power BI to analyze electricity consumption anomalies, identify operational risks, and generate actionable insights.
